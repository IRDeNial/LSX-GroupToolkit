Leak.sx Group Leader Toolkit

A toolkit to help group leaders manage their groups

View project onGitHub

About

This is a userscript that was developed for group leaders on a website known as Leak.sx who easily want to manage their group. The current group management page is a bit... Under-featured.

How to Install

  • Install either need Greasemonkey for Mozilla FireFox or Tampermonkey for Google Chrome
  • Visit https://github.com/IRDeNial/LSX-GroupToolkit/raw/master/grouptoolkit.user.js to install the userscript.
  • Visit the normal group management page (managegroup.php?gid=)

Want to contribute?

Clone the repository and once you're done, create a pull request and I'll review what you have and look into adding your code! Just visit the GitHub Page and go from there!
